# 🚀 快速启动指南

恭喜您！畅想教师教学工具项目已经创建完成。接下来让我们开始开发之旅！

## 📋 第一步：安装依赖

打开终端，进入项目目录：

```bash
cd e:\Code\ChangXiangTeacher
npm install
```

## 🗄️ 第二步：初始化数据库

运行以下命令创建数据库和示例数据：

```bash
npm run db:init
```

这将在您的用户目录下创建数据库文件。

## 🔧 第三步：启动开发环境

```bash
npm run dev
```

这将同时启动：
- Electron 主进程
- Vite 开发服务器
- Vue 应用

## 🔍 调试技巧

1. **查看数据库**：使用 DB Browser for SQLite 打开数据库文件
2. **开发者工具**：按 `Ctrl+Shift+I` 打开开发者工具
3. **热重载**：修改代码后自动刷新

## 🆘 遇到问题？

1. 检查 Node.js 版本是否 >= 18.0
2. 确保所有依赖安装成功
3. 查看控制台错误信息
4. 参考 `DEVELOPMENT.md` 中的故障排除部分

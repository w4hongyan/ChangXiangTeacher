## 📦 快速开始

### 环境要求
- Node.js 18.0 或更高版本
- npm 或 yarn 包管理器

### 安装依赖
```bash
# 进入项目目录
cd e:\Code\ChangXiangTeacher

# 安装依赖
npm install

# 初始化数据库
npm run db:init

# 开发环境启动
npm run dev

# 构建生产版本
npm run build
```

### 开发调试
```bash

npm run dev

# - Vite开发服务器：http://127.0.0.1:3000

# 数据库文件位置：
# C:\Users\Administrator\AppData\Roaming\changxiang-teacher\database.db
```

```javascript

// 当前环境: C:\Users\Administrator\AppData\Roaming\changxiang-teacher\database.db
// macOS: ~/Library/Application Support/changxiang-teacher/database.db
// Linux: ~/.config/changxiang-teacher/database.db

// 数据库初始化

// 已创建的数据表：
```

## 🐛 问题反馈和调试

### 常见问题解决

#### 学生列表不显示数据
1. 检查开发者工具控制台是否有错误
2. 验证数据库是否已初始化：`npm run db:init`
3. 检查IPC通信是否正常：在控制台运行 `window.electronAPI.students.list()`
4. 确认Pinia状态管理是否正常工作

#### 应用启动问题
1. 确认Node.js版本 >= 18.0
2. 清除依赖重新安装：`rm -rf node_modules && npm install`
3. 检查端口是否被占用（3000, 3001）

#### 数据库问题
1. 数据库位置：`C:\Users\Administrator\AppData\Roaming\changxiang-teacher\database.db`
2. 重新初始化：删除数据库文件后运行 `npm run db:init`
